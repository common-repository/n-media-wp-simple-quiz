== Description ==
Introducing Easy Quiz Maker with WordPress, the simplest way to create unlimited quizzes with multiple choice questions. With a user-friendly interface, you can easily set up questions with multiple answers and designate the correct answer for each question, as well as assign a total number of marks. Customize the look and feel of your quizzes to match your website's style, and tailor the messages users receive after submitting their quiz.
Results are saved as a custom post type, including all relevant information such as total marks, obtained marks, and correct answers. This information is easily accessible and can be used to track progress and analyze results over time.

== Advanced Features in PRO ==
For even more advanced features, upgrade to the PRO version for access to quiz stats and results, the ability to set up quizzes for specific user roles, show results to users, and more. With the PRO version, you can also set time limits for each question, redirect users to a specific URL after completing the quiz, and send results via email directly from the result page. Keep track of time elapsed and see results displayed in a graph, making it easy to analyze and track progress. With Easy Quiz Maker with WordPress, creating and managing quizzes has never been easier!

== Installation ==

1. Upload plugin directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. After activation, you can create Quiz from `WP Simple Quiz` menu
